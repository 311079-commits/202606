# 💱 外幣匯率到價通知系統

## 🌟 系統概述

這是一個現代化的金融科技網頁應用程式（SPA），提供用戶匯率變動通知功能，並為管理員提供實時數據分析儀表板。

## 🚀 核心功能

### 1️⃣ Google 帳戶整合
- 使用 Google Identity Services (GIS) API
- 自動解碼並提取用戶信息
- 本地存儲登入狀態

### 2️⃣ 使用者功能
- **匯率監控表單**：支援 9 種主要貨幣
  - USD (美元) 🇺🇸
  - JPY (日圓) 🇯🇵
  - EUR (歐元) 🇪🇺
  - CNY (人民幣) 🇨🇳
  - KRW (韓圜) 🇰🇷
  - HKD (港元) 🇭🇰
  - GBP (英鎊) 🇬🇧
  - VND (越南盾) 🇻🇳
  - TWD (台幣) 🇹🇼

- **條件選擇**：「低於」或「高於」
- **歷史紀錄**：查看所有已設定的通知

### 3️⃣ 管理員後台 (需用指定 Email 登入)
- **圓餅圖分析**：全站最熱門訂閱貨幣比例
- **統計卡片**：
  - ⏳ 待送出通知數
  - ✅ 已送出通知數
  - 👥 總訂閱數
- **歷史紀錄表格**：所有用戶的訂閱記錄

### 4️⃣ API 集成
- **POST**：將表單數據發送到 Google Apps Script (GAS)
- **GET**：從 GAS 讀取試算表數據進行分析

## 🎮 使用方式

### 普通使用者
1. 點擊「Sign in with Google」登入
2. 在表單中選擇要監控的貨幣
3. 設定通知條件（低於/高於）
4. 輸入目標匯率
5. 點擊「✨ 設定通知」提交
6. 在「📜 我的歷史設定」中查看所有訂閱

### 管理員
1. 用符合 `adminEmail` 的帳戶登入
2. 點擊「👑 管理員後台」按鈕
3. 查看：
   - 📊 各貨幣訂閱分佈圖表
   - 📋 全站訂閱統計數據
   - 📋 所有使用者的訂閱紀錄

## 🔐 安全建議

1. **Google Client ID**：在生產環境中設定授權域名
2. **GAS URL**：確保只接受來自您的域名的請求
3. **Email 驗證**：建議後端驗證用戶 Email
4. **HTTPS**：生產環境必須使用 HTTPS

## 🐛 故障排除

### Google Sign-In 不顯示
- ✅ 檢查 `GOOGLE_CLIENT_ID` 是否正確
- ✅ 確認授權域名已新增到 Google Cloud Console

### GAS 連接失敗
- ✅ 檢查 `GAS_URL` 是否正確
- ✅ 查看瀏覽器控制台的錯誤訊息
- ✅ 確認 GAS 部署為「網頁應用程式」

### 管理員後台不顯示
- ✅ 驗證登入的 Email 與 `adminEmail` 完全相符
- ✅ 清除本地存儲後重新登入：`localStorage.clear()`
